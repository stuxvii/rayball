rayball is a reimplementation of the H*xBall client with biased additions, built using raylib.

### features
- [x] server list
- [x] in-game clock
- [ ] server joining
  - [x] websocket handshake
  - [ ] encode data
  - [ ] send data
  - [ ] webrtc
  - [ ] parse render data
  - [ ] probably more things
- [x] settings save
- [x] better ui
- [x] better ux
- [x] nav bar
