rayball is a very early work in progress reimplementation of the H\*xBall client with biased additions, built using raylib.

### features
- [x] server list
- [x] in-game clock
- [ ] server joining
  - [ ] websocket handshake
  - [ ] password check
  - [ ] webrtc
  - [ ] probably more things
- [x] settings save
- [x] better ui
- [x] better ux
- [x] nav bar
