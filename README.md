rayball was supposed to be a reimplementation of the HaxBall client with biased additions, built using raylib.

however, due to [a github comment](https://github.com/haxball/haxball-issues/issues/1805) made by basro, the owner, dev, and author of haxball i have decided to stop development.

besides ui/ux, this only has a decompilation of the A class in the game's source and a function to read the room list.

i really wanted to keep developing this but ig not everyone can have all.
